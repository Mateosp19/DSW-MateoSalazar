Atributos y valores

Los atributos en HTML son palabras especiales que se utilizan dentro de la etiqueta de apertura, esto, paara controlar el comportamiento de un elemento. A pesar de que cada una de las etiquetas HTML define sus propios atributos, existen algunos atributos comunes a casi todas las etiquetas, los cuales se pueden dividir en cuatro grupos según su funcionalidad:

  • Atributos básicos: Se utilizan en la mayoria de etiquetas HTML y XHTML, aunque tienen mayor sentido cuando se hace uso de hojas de estilo en cascada (CSS)
  • Atributos de internacionalización: Son utilizados en páginas que muestran contenidos en varios idiomas y las que deseen indicar de forma explícita el idioma de sus contenidos
  • Atributos de eventos: Son utilizados en las páginas web que incluyen código JavaScript para realizar acciones dinámicas sobre los elementos de la página.
  • Atributos de foco: Se denominan atributos de foco cuando un elemento del documento ha sido seleccionado.
